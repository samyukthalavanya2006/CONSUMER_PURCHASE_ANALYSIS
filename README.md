# Consumer Purchase Insights Dashboard

## Project Overview
This project is an end-to-end Data Analytics project built using **Python, PostgreSQL, and Power BI** to analyze customer purchase behavior and generate meaningful business insights.

The project focuses on customer spending patterns, subscription behavior, product performance, revenue analysis, and review ratings using data cleaning, SQL querying, and interactive dashboarding.

---

## Tools & Technologies Used
- Python (Pandas, NumPy)
- PostgreSQL
- Power BI
- VS Code
- Jupyter Notebook

---

## Project Workflow
1. Data Cleaning & Preprocessing using Python
2. Missing Value Handling
3. Feature Engineering
4. Data Upload to PostgreSQL
5. SQL-Based Business Analysis
6. Interactive Dashboard Creation in Power BI

---

## Key Features
- Revenue Analysis
- Customer Segmentation
- Subscription Insights
- Product Performance Analysis
- Review Rating Analysis
- Shipping Type Insights
- Age Group Analysis

---

## SQL Concepts Used
- GROUP BY
- Aggregate Functions
- CASE WHEN
- CTEs
- ROW_NUMBER()
- Window Functions
- Subqueries

---

## Python Tasks Performed
- Missing value handling
- Feature engineering
- Data transformation
- Exploratory Data Analysis (EDA)

---

## Power BI Dashboard Includes
- KPI Cards
- Revenue by Category
- Sales by Category
- Customer Subscription Analysis
- Revenue by Age Group
- Interactive Filters & Slicers
